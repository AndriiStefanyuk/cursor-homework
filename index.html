Команди Git

git config --global user.name вводимо ім'я користувача
git config --global user.email вводимо пошту до github
git init ініціалізуємо репозиторій
git status перевірка змін в файлах
git add додає вибрані файли
git commit зберігає поточний зліпок проекту
git log дозволяє переглянути всі коміти з їхніми ідентифікаторами
git checkout дає модливість переключатись між гілками чи комітами
git branch показує існуючі гілки
git push завантажує закомічені файли в глобальний репозиторій
git pull заваютажує файли з віддаленого репозиторію на локальний
git merge дозволяє обєднати гілки
git clone робить копію репозиторію

.gitignore приховуєвід гіта вказані в ньому файли/папки

git revert дозволяє відмінии коміт
git reset дає можливість видалити коміт і його історію з гілки
git rebase об'єднує гілки, копіюючи набори комітів з однієї в другу
git cherry-pick копіює прописані коменти в head
git stash тимчасво зберігає всі незафіксовані зміни відслідковуваних файлів

Послідовність команд для додавання дз на віддалений репозитрій

git init виконується один раз
git add .
git commit -m "first commit"
git remote add origin https://github.com/AndriiStefanyuk/cursor-homework.git
git push -u origin master
